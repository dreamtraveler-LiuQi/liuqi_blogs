JS的Object.defineProperty用法

发表于 2021-08-06 更新于 2022-02-24 分类于 javascript 阅读次数： 本文字数： 2.5k 阅读时长 ≈ 2 分钟

Object.defineProperty 用于在一个对象上定义一个新的属性，或者修改一个对象现有的属性，并返回这个对象。

属性描述符

属性默认值说明
configurablefalse描述属性是否可以被删除，默认为 false
enumerablefalse描述属性是否可以被 for…in 或 Object.keys 枚举，默认为 false
writablefalse描述属性是否可以修改，默认为 false
getundefined当访问属性时触发该方法，默认为 undefined
setundefined当属性被修改时触发该方法，默认为 undefined
valueundefined属性值，默认为 undefined
